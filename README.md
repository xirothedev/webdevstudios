## Savi E-commerce Fullstack Turborepo

Monorepo cho dự án e-commerce sử dụng kiến trúc fullstack hiện đại.

### Apps chính

- `apps/web`: Frontend dùng Next.js (App Router), Tailwind CSS và motion/react cho animation.

### FeaturesGrid

- Component `FeaturesGrid` trong `apps/web` hiển thị bộ sưu tập merch WebDev Studios:
  - Áo thun WDS, huy hiệu, dây đeo (lanyard) và pad chuột limited.
  - Glassmorphism cards, parallax/tilt hiệu ứng 3D nhẹ.
  - Ảnh hero sản phẩm áo thun WDS với kích thước lớn, glow mạnh và hover scale/rotate nổi bật.

### Dev notes

- Chạy lệnh lint/format được khuyến nghị:
  - `pnpm lint`
  - `pnpm format` (hoặc cấu hình tương đương trong repo)
